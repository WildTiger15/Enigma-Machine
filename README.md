# Enigma-Machine
Enigma machine

Description:

  This project contains Java implementations of several cryptographic ciphers. The goal is to demonstrate encryption and          decryption methods using classic ciphers in Java.

Ciphers Included:

Affine Cipher: Implements the Affine cipher which uses mathematical operations to encrypt and decrypt messages.

Caesar Cipher: A substitution cipher where each letter in the plaintext is shifted a certain number of places down the alphabet.

Enigma Cipher: Simulates the Enigma machine used during World War II.

Rotor Cipher: A more complex encryption method that uses a rotating substitutive mechanism.

CipherBase: An abstract base class for common cipher operations.
